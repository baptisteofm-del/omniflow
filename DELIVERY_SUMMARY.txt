TECHNICAL SPECIFICATIONS
================================================================================

STACK:
  • Frontend: React 18 + Next.js 14 + TypeScript
  • Backend: Next.js API Routes
  • Database: Supabase PostgreSQL
  • Storage: Supabase Storage (S3-compatible)
  • UI Framework: Tailwind CSS
  • Icons: Lucide React

BROWSER SUPPORT:
  ✅ Chrome/Edge 120+
  ✅ Firefox 121+
  ✅ Safari 17+
  ✅ Mobile Safari (iOS 17+)
  ✅ Chrome Mobile (Android)

PERFORMANCE:
  • Avatar upload: < 3 seconds for 1MB image
  • Page load: < 2 seconds with 100+ models
  • Stats query: < 500ms response time
  • Bundle size impact: +33 KB (gzipped: +8 KB)

STORAGE:
  • Avatar path: avatars/{agencyId}/{modelId}.{ext}
  • Max file size: 5MB per avatar
  • Supported formats: JPEG, PNG, WebP, GIF
  • CDN delivery: Via Supabase Storage CDN
