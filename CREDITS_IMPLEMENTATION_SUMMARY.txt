🔑 CARACTÉRISTIQUES CLÉS
═══════════════════════════════════════════════════════════════════════════════

SYSTÈME UNIFIÉ
  • 1 crédit = 1 unité (IA génération ou trend Instagram)
  • 1 RUN = 10 crédits = 9€ (fixe, standard)
  • Applicable à tous les features IA
  • Valides indéfiniment

AUTO TOP-UP
  • Déclenchement automatique au-dessous d'un seuil configurable
  • Montant d'achat configurable (multiple de 10)
  • Trigger SQL natif (exécution instantanée, pas de polling)
  • Transaction enregistrée dans l'historique

CODES PROMO
  • Types: percent (%), fixed (€), ou credits (bonus)
  • Applicable à: abonnement, crédits, ou les deux
  • Limites: max uses globales, max par utilisateur, expiration
  • Spécifique aux plans (optionnel)
  • Montant minimum (optionnel)

ANTI-ABUS
  • 1 utilisation max par email/code (configurable)
  • Limites d'utilisation globales
  • Vérification d'expiration
  • Suivi complet dans promo_code_uses

SÉCURITÉ
  • RLS sur toutes les tables
  • Policies restreignent par agence propriétaire
  • Auth via createClient() à chaque route
  • Vérification rôle admin pour CRUD promo

TRAÇABILITÉ
  • Historique complet: date, type (purchase/consumption/bonus), montant, solde après
  • Feature taggée (ai_generation, trend_run, etc.)
  • Payment ID et promo code enregistrés
